Portfolio personal de Olai Garcia Villar orientado a perfil Backend Java Developer, con experiencia en proyectos del sector bancario.

## Perfil profesional

Desarrollo backend con Java y ecosistema Spring en entornos corporativos de alta exigencia.
Especializacion en:

- Procesos batch
- Integracion con bases de datos relacionales
- Mantenimiento evolutivo de aplicaciones backend

## Contenido del portfolio

- Formacion academica
- Experiencia profesional
- Proyectos destacados
- Contacto rapido y redes

## Tecnologias del proyecto

- HTML5
- CSS3
- JavaScript
- Font Awesome (iconografia)

## Estructura principal

- index.html: pagina principal del portfolio
- CSS/: estilos por secciones
- JS/script.js: comportamiento e interacciones de la pagina
- Resources/images-app/: imagenes del portfolio
- Resources/icons-app/: recursos graficos adicionales
- CV-OlaiGV.pdf: CV descargable

## Como ejecutar en local

1. Abre la carpeta MyPortfolio en VS Code.
2. Ejecuta index.html con Live Server o abrelo directamente en el navegador.
3. Revisa responsive en desktop y movil.
